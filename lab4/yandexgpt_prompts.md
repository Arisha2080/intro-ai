## Промпт 1: Генерация заголовка

### Текст промпта:
```
На основе предоставленного описания товара создай привлекательный маркетинговый заголовок.

Требования к заголовку:
- Длина: 40-60 символов
- Должен быть цепляющим и запоминающимся
- Отражать ключевую ценность продукта
- Использовать эмоциональные триггеры

Описание товара: {текст_товара}

Создай 3 варианта заголовка.
```

### Рекомендуемые параметры YandexGPT:
- **Temperature (Температура):** 0.7
- **Max tokens (Длина ответа):** 150
- **Top P:** 0.9

### Пример результата:
```
Вариант 1: "ArtMaster Pro X1: Профессиональное творчество без границ"
Вариант 2: "Воплоти свои идеи в цифре - ArtMaster Pro X1"
Вариант 3: "Точность художника, скорость профессионала"
```

## Промпт 2: Создание маркетингового описания

### Текст промпта:
```
Создай продающее маркетинговое описание товара для интернет-магазина.

Требования:
- Длина: 150-200 слов
- Структура: вступление, ключевые преимущества, призыв к действию
- Тон: профессиональный, но дружелюбный
- Использовать списки для улучшения читаемости
- Подчеркнуть уникальные особенности

Описание товара: {текст_товара}

Заголовок для использования: {выбранный_заголовок}
```

### Рекомендуемые параметры YandexGPT:
- **Temperature (Температура):** 0.6
- **Max tokens (Длина ответа):** 400
- **Top P:** 0.85

### Пример результата:
```
ArtMaster Pro X1: Профессиональное творчество без границ

Откройте новые горизонты цифрового искусства с графическим планшетом ArtMaster Pro X1.
Это не просто инструмент – это продолжение руки художника, передающее каждое движение
с беспрецедентной точностью благодаря 8192 уровням чувствительности к нажатию.

Ключевые преимущества:
• Профессиональный стилус с технологией распознавания наклона до 60°
• Рабочая область 13.3" Full HD для комфортной работы над деталями
• Время отклика 10 мс – рисуйте без задержек
• 8 программируемых клавиш и сенсорное кольцо для быстрого доступа
• Премиум алюминиевый корпус толщиной всего 8 мм
• Полная совместимость с Windows, macOS и профессиональным ПО

ArtMaster Pro X1 создан для художников, дизайнеров и иллюстраторов, которые не идут
на компромиссы. Работайте в Photoshop, Illustrator, Clip Studio Paint с ощущением
рисования на настоящей бумаге.

Воплотите свои идеи в жизнь! Закажите ArtMaster Pro X1 и получите набор сменных наконечников
для стилуса в подарок при оформлении заказа сегодня.
```
